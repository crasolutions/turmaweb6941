Sites totalmente responsivos

Devido a grande evolução na área de tecnologia contamos com diversos dispositivos para acessar a internet. Imagina se fossemos ter que criar diversos códigos ou formatos para um site abrir de uma forma acessível em todos dispositivos disponíveis no mercado?

Daí vem o conceito e a técnica chamada de design responsivo. A ideía é que o site se ajuste a qualquer dispositivo que o usuário estiver usando. Hoje não se pode ter um site só para desktop, ou só mobile mas sim um site que seja responsivo e se encaixe a necessidade do usuário tornando a navegação fácil e inteligente
